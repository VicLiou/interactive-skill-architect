# 自我測試成績單模板 (Eval Report Template)

# 這是 architect 在自我測試模式 Phase E3 輸出成績單的統一格式（單一來源）。
# eval-mode.md Phase E3 一律引用此模板，不另抄一份。填好的範例見 assets/examples/example-eval-report.md。

## 執行摘要
- 一段話總結：本次跑了幾個案例、整體通過率、有無安全類紀律破戒（若有，此處先點名）。

## 統計摘要
| 指標 | 數值 | 說明 |
|------|------|------|
| 執行案例數 | {{n}} | 選定範圍內實際重放的案例 |
| 全數通過 | {{n}} | 機械項＋語意項皆 PASS |
| 有破戒 | {{n}} | 任一項 FAIL |
